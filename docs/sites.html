05 — Method

Built on free, public Earth-observation data.

Topography

Rainfall

UCSB CHC · 2016–2025

Vegetation

sentinel-2-l2a · 2017–2026

Caveats

  • CHIRPS 5km exceeds the field size, so both blocks receive the same rainfall value — accurate for regional climate, not for in-field variation.
  • Slope is computed with a numerical gradient on 30m DEM pixels — appropriate for general terrain characterisation, not for drainage-engineering layouts.
  • NDVI/NDMI annual trends from this dataset are biased by uneven cloud-clear scene availability and should not be read as canopy decline.
